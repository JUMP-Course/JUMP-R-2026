## 2. 研究问题

在美国≥20岁成年人中，超重和肥胖是否与高血压患病风险独立相关，核心分析采用多因素Logistic回归，在调整年龄、性别、种族、教育水平后，评估不同BMI分类（正常体重/超重/肥胖）与高血压患病状态之间的关联。

## 3. 数据来源

NHANES 2021-2023数据，公开数据，可以上传。

## 4. 主要变量

- 结局变量：高血压患病状态（是/否）
- 暴露变量/预测变量：BMI分类（正常体重/超重/肥胖）
- 协变量：年龄、性别、种族和教育水平
