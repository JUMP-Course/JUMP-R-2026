# 个人数据分析项目 - README 初稿

## 一、项目选题
2017-2023年美国成年人甲型肝炎抗体阳性率变化趋势及人口学影响因素分析

## 二、选题是否合适（课程重点）
1. 题目：聚焦甲肝阳性率 + 人口学影响因素，范围明确、可完成
2. 数据可得：使用 NHANES 公开数据，已下载并整理完成
3. 变量清晰：结局变量（LBXHA）、协变量（年龄、性别、教育、收入、种族）均明确
4. 6 周可完成：描述统计 + 分组比较 + 简单回归，属于最小可完成分析闭环

## 三、数据来源
美国国家健康与营养检查调查（NHANES）
2017-2020 和 2021-2023 两个周期数据

## 四、分析计划（最小闭环）
1. 数据清洗：删除缺失、筛选变量
2. 描述统计：样本量、阳性率分布
3. 分组比较：不同年龄、性别、教育的阳性率差异
4. 总结结论
