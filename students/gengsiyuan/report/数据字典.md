数据字典（基于NHANES数据库的尿镉与桥本甲状腺炎关联分析）

研究主题：尿镉暴露与桥本甲状腺炎的关联性分析
数据来源：NHANES 数据库
研究人群：年龄≥20 岁成年人
一、标识变量
SEQN：受访者唯一编号
二、暴露变量
URXUCD：尿镉浓度，μg/L（连续变量）
URXUCR：尿肌酐，mg/dL（用于校正尿镉）
三、结局变量
LBXTPO：甲状腺过氧化物酶抗体，IU/mL，临界值 9
LBXATG：甲状腺球蛋白抗体，IU/mL，临界值 4
hashimoto：桥本甲状腺炎（衍生变量），1 = 阳性（LBXTPO>9 或 LBXATG>4），0 = 阴性
四、协变量
RIDAGEYR：年龄（岁）
RIAGENDR：性别，1 = 男，2 = 女
RIDRETH1：种族，1 = 墨西哥裔，2 = 其他西班牙裔，3 = 非西裔白人，4 = 非西裔黑人，5 = 其他
DMDEDUC2：教育水平，1=＜9 年级，2=9-11 年级，3 = 高中毕业，4 = 大学及以上
INDFMPIR：收入贫困比值（0~5）
BMXBMI：体质指数，kg/m²
SMQ020：吸烟史，1 = 吸烟超 100 支，2 = 否
BPQ020：高血压病史，1 = 是，2 = 否
五、权重变量
WTMEC2YR：两年抽样权重
wt_6yr：六年合并权重，wt_6yr = WTMEC2YR / 3
六、补充说明
桥本甲状腺炎由两项甲状腺抗体联合判定；
尿镉浓度需经尿肌酐校正；
六年权重用于多周期合并数据分析。
