数据字典（基于 NHANES 数据库的尿镉与桥本甲状腺炎关联分析）
研究主题：尿镉暴露与桥本甲状腺炎的关联性分析
数据来源：NHANES 2007-2008、2009-2010、2011-2012 三期合并数据库
研究人群：年龄 ≥ 20 岁成年人
一、标识变量
SEQN：受访者唯一编号
二、暴露变量
URXUCD：原始尿镉浓度，μg/L（连续变量）
URXUCR：尿肌酐，mg/dL（用于校正尿镉）
URXUCD_cr：肌酐校正后尿镉浓度（衍生连续变量）
计算公式：URXUCD_cr = URXUCD / URXUCR * 1000
注：公式与 R 分析代码保持一致，单位换算参照 NHANES 官方数据文档。
三、结局变量
LBXTPO：甲状腺过氧化物酶抗体，IU/mL，临界值 9
LBXATG：甲状腺球蛋白抗体，IU/mL，临界值 4
hashimoto：桥本甲状腺炎（衍生二分类变量）
赋值规则：
1（阳性）：LBXTPO > 9 或 LBXATG > 4
0（阴性）：LBXTPO ≤ 9 且 LBXATG ≤ 4（两项指标均有有效结果且未超标）
NA（缺失）：两项抗体均无检测结果；或仅单一项指标有效且未超标，另一项缺失
四、原始协变量
RIDAGEYR：年龄（岁）
RIAGENDR：性别，1 = 男，2 = 女
RIDRETH1：种族，1 = 墨西哥裔，2 = 其他西班牙裔，3 = 非西班牙裔白人，4 = 非西班牙裔黑人，5 = 其他
DMDEDUC2：教育水平，1 = ＜9 年级，2 = 9‑11 年级，3 = 高中毕业，4 = 大学及以上
INDFMPIR：收入贫困比值（0~5）
BMXBMI：体质指数，kg/m²
SMQ020：吸烟史，1 = 吸烟超 100 支，2 = 否
BPQ020：高血压病史，1 = 是，2 = 否
五、分类转换后协变量（衍生因子变量）
sex：性别（分类变量），取值：男、女
映射规则：RIAGENDR = 1 → 男；RIAGENDR = 2 → 女
race：种族（分类变量），取值：白人、黑人、西班牙裔、其他
映射规则：
白人：RIDRETH1 = 3（非西班牙裔白人）
黑人：RIDRETH1 = 4（非西班牙裔黑人）
西班牙裔：RIDRETH1 = 1（墨西哥裔）、2（其他西班牙裔）
其他：RIDRETH1 = 5
education：教育水平（分类变量），取值：低于高中、高中、大学及以上
映射规则：
低于高中：DMDEDUC2 = 1、2
高中：DMDEDUC2 = 3
大学及以上：DMDEDUC2 = 4、5
smoke：吸烟史（分类变量），取值：从不吸烟、吸烟
映射规则：SMQ020 = 2 / 文本值 No → 从不吸烟；SMQ020 = 1 / 文本值 Yes → 吸烟
hypertension：高血压病史（分类变量），取值：有、无
映射规则：BPQ020 = 1 / 文本值 Yes → 有；BPQ020 = 2 / 文本值 No → 无
六、权重变量
WTMEC2YR：两年抽样权重
wt_6yr：六年合并权重（衍生变量）
计算公式：wt_6yr = WTMEC2YR / 3
七、数据清洗与统一规则
特殊编码转换：将原始数据中代表缺失、拒答的数值 777、888、999 统一转换为系统缺失值 NA。
人群筛选：剔除年龄 RIDAGEYR ＜ 20 岁样本，仅保留成年人群。
缺失值处理：采用完整病例分析法，剔除以下任一变量存在缺失的样本：校正尿镉、桥本甲状腺炎结局、六年合并权重、全部人口学及临床协变量。
异常值处理：采用四分位距法（IQR）识别连续变量异常值，仅用于描述统计，不做剔除，保留原始数据分布。
数据合并流程
分周期读取人口学、甲状腺指标、尿镉、体格检查、吸烟、血压等数据表；
同类型数据表按年度纵向合并；
以 SEQN 为关联字段，横向左连接整合所有数据表。

文件名称：NHANES_2007_2012_urine_cadmium_final_clean.xlsx
包含内容：全部原始变量与衍生变量
