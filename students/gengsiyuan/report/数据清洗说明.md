数据清洗说明（NHANES 尿镉与桥本甲状腺炎关联分析）
一、数据来源与周期
数据库：NHANES 2007-2008、2009-2010、2011-2012 三个调查周期
原始数据表：DEMO（人口学数据）、THYROD（甲状腺抗体）、UHM（尿镉及尿肌酐）、BMX（体格测量）、SMQ（吸烟相关信息）、BPQ（高血压相关信息）
二、数据合并流程
纵向堆叠：使用bind_rows函数，将三个周期内同类型数据表按行合并。
横向合并：以受访者唯一编号SEQN为关联键，通过左连接依次整合数据集，合并顺序：尿镉数据 → 甲状腺指标 → 人口学数据 → 体格测量 → 吸烟信息 → 高血压信息。
三、缺失值与特殊编码处理
将原始数据中代表拒答、未知、缺失的特殊编码777、888、999统一转换为系统缺失值NA。转换完成后统计关键变量缺失率，并绘制缺失率条形图完成可视化评估。
四、样本筛选步骤
基础人群筛选：保留RIDAGEYR ≥ 20岁成年人，剔除未成年人。
完整病例筛选：采用完整病例分析法，剔除以下任一变量存在缺失的样本：
暴露变量：URXUCD_cr（肌酐校正后尿镉）
结局变量：hashimoto（桥本甲状腺炎）
权重变量：wt_6yr（六年合并权重）
协变量：RIDAGEYR、sex、race、education、smoke、hypertension、BMXBMI、INDFMPIR
最终得到无缺失值的分析数据集。
五、变量衍生与重编码规则
暴露变量校正
衍生变量：URXUCD_cr（肌酐校正尿镉，连续变量）
计算公式：URXUCD_cr = URXUCD / URXUCR * 1000
结局变量定义
衍生变量：hashimoto（桥本甲状腺炎，二分类变量）
阳性（1）：LBXTPO > 9 或 LBXATG > 4（任一抗体超标，另一抗体可缺失）
阴性（0）：LBXTPO ≤ 9 且 LBXATG ≤ 4（两项抗体均有有效结果且未超标）
缺失（NA）：两项抗体均无检测结果；或仅单一项指标有效且未超标，另一项缺失
分类协变量映射（原始变量 → 衍生变量）
性别（sex）：RIAGENDR = 1→男；RIAGENDR = 2→女
种族（race）：RIDRETH1 = 3→白人；RIDRETH1 = 4→黑人；RIDRETH1 = 1/2→西班牙裔；RIDRETH1 = 5→其他
教育水平（education）：DMDEDUC2 = 1/2→低于高中；DMDEDUC2 = 3→高中；DMDEDUC2 = 4/5→大学及以上
吸烟史（smoke）：SMQ020 = 2/ 文本No→从不吸烟；SMQ020 = 1/ 文本Yes→吸烟
高血压病史（hypertension）：BPQ020 = 1/ 文本Yes→有；BPQ020 = 2/ 文本No→无
抽样权重计算
衍生变量：wt_6yr（六年合并权重）
计算公式：wt_6yr = WTMEC2YR / 3
六、异常值识别与处理
采用四分位距法（IQR）识别异常值，将数值超出 Q1 - 1.5×IQR 或 Q3 + 1.5×IQR 的观测值判定为异常值。本研究仅统计异常值数量与占比、用于描述分析，不剔除异常样本，保留数据原始分布特征。
七、质量控制步骤
使用identical函数校验不同周期下同数据表的列名一致性；
核查合并后数据集的SEQN，确保无重复样本；
计算各数据表样本匹配率，评估数据整合质量；
绘制尿镉浓度对数分布直方图，分析变量分布形态。
八、最终数据集输出
文件名称：NHANES_2007_2012_urine_cadmium_final_clean.xlsx
文件格式：Excel（.xlsx）
数据集说明：包含全部原始变量与衍生变量，样本无缺失值；
样本量：以 R 代码运行输出结果n_after为准。
九、所用软件与工具包
分析软件：R 语言（4.0 及以上版本）
核心工具包：nhanesA（数据提取）、tidyverse（数据清洗与可视化）、survey（加权统计分析）、naniar（缺失值可视化）、writexl（数据导出）
