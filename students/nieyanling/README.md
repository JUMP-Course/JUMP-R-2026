2.项目题目

膳食添加糖摄入与成人血脂异常的关联分析

3.研究问题

本研究以美国NHANES 2017-2018年成人数据为基础，探究膳食添加糖摄入与成人血脂异常的关联，具体研究问题包括：
1.  描述不同添加糖摄入水平人群的血脂指标分布特征；
2.  分析添加糖摄入水平与血脂异常患病风险之间的关联；
3.  探索添加糖摄入水平与血脂异常的剂量-反应关系。

4.数据来源

美国NHANES 2017-2018年成人数据

5.主要变量
暴露变量
膳食添加糖摄入：使用NHANES膳食数据中的总添加糖摄入量（单位：g/天），按三分位数分为低、中、高摄入组。

结局变量
血脂异常指标：
  1. 总胆固醇（TC）：≥240 mg/dL定义为高胆固醇
  2. 甘油三酯（TG）：≥200 mg/dL定义为高甘油三酯
  3. 高密度脂蛋白（HDL）：男性＜40 mg/dL、女性＜50 mg/dL定义为低HDL
  4. 低密度脂蛋白（LDL）：≥160 mg/dL定义为高LDL
