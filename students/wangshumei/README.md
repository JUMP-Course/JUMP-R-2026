# 王淑玫的 R 语言课程项目

## 项目题目

肿瘤分级与宫颈癌患者总生存期的关联

## 研究问题

基于TCGA-CESC临床数据，分析不同肿瘤分级宫颈癌患者的总生存差异，评估肿瘤分级是否为宫颈癌预后的独立影响因素。

## 研究背景

宫颈癌（cervical cancer）：也称为子宫颈癌，是女性生殖道恶性肿瘤，最常见类型为鳞状细胞癌。病因主要为人乳头状瘤病毒（HPV）感染。宫颈癌的高危人群为有多个性伴侣者、性生活初夜早和免疫功能缺陷者。宫颈癌在早期可能无明显症状，随着疾病进展，可能出现阴道流血、异常阴道排液以及肿瘤侵犯到邻近周围的组织或器官时出现的症状。


宫颈癌的分级（grading）：反应肿瘤细胞恶性程度

G1（高分化）：癌细胞接近正常宫颈细胞，生长较慢，侵袭性较低。
G2（中分化）：介于高分化和低分化之间，恶性程度中等。
G3（低分化）：癌细胞原始且异常，增殖活跃，易转移，预后较差。


## 研究内容

暴露因素：肿瘤分级（G1 / G2 / G3）

结局：总生存（OS）——从确诊到死亡或末次随访时间

协变量：年龄、FIGO 分期、治疗方式（手术 vs 非手术）、术后残留病灶（有/无）


具体内容

1.描述性分析：比较不同分级组（G1\G2\G3）在年龄、分期、治疗方式等的基线特征比较

2.Kaplan-Meier生存曲线：绘制G1/G2/G3三组生存曲线，Log-rank检验比较三组生存曲线是否有统计学差异

3.单因素Cox回归：筛选与总生存期相关的变量（p<0.1）。逐个检验每个变量（分级、年龄、分期、治疗方式、残留病灶）与总生存的关系

4.多因素Cox回归：以肿瘤分级为主要暴露，校正协变量，评估独立关联，结果用风险比（HR）和95%置信区间表示

5.森林图：可视化展示各变量的风险比（HR）及95%置信区间
