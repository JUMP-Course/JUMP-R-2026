# 基于 NHANES 数据的成年人 BMI 与高血压患病风险关联分析

## 2. 项目主题方向
慢病危险因素分析

## 3. 研究背景
高血压是常见慢性病之一，也是心脑血管疾病的重要危险因素。肥胖与高血压之间可能存在密切关联，并且肥胖属于可干预的健康危险因素。因此，本项目拟利用公开健康调查数据，探索成年人肥胖与高血压患病风险之间的关系。

## 4. 研究问题
肥胖人群发生高血压的风险是否高于正常体重人群？

## 5. 数据来源
本项目拟使用 NHANES 公开数据库。NHANES 是美国国家健康与营养调查数据库，包含人口学信息、身体测量、血压测量、疾病问卷和生活方式信息等内容。

## 6. 研究对象
18岁及以上成年人。

## 7. 主要变量

### 暴露变量
BMI，拟分为正常体重、超重和肥胖。

### 结局变量
是否患有高血压。拟根据血压测量值、自报医生诊断高血压和降压药使用情况综合定义。

### 协变量
年龄、性别、吸烟状态。

## 8. 初步分析计划
1. 读取并整理 NHANES 数据；
2. 完成变量筛选和数据清洗；
3. 描述研究对象基本特征；
4. 绘制 BMI 分布图；
5. 绘制不同 BMI 分组的高血压患病率图；
6. 使用 Logistic 回归分析 BMI 与高血压之间的关联；
7. 报告 OR 值、95%CI 和 P 值。
