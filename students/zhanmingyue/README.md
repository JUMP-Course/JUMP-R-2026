\## 项目题目



腰围与糖尿病风险的关联分析



\## 研究问题



校正年龄、性别和总胆固醇后，腰围是否与糖尿病（糖化血红蛋白≥6.5%）独立相关？



\## 数据来源



Kaggle公开数据集（包含glyhb、waist、age、gender、chol等变量，样本量404，部分缺失）

#第四次课（基线特征表）
1.对腰围、年龄、总胆固醇进行正态性检验（Shapiro-Wilk），发现均不服从正态分布（p<0.05）。
2.使用 `gtsummary` 制作 Table 1，连续变量以中位数（四分位距）描述，分类变量以频数（百分比）描述。
3.组间比较采用 Wilcoxon 秩和检验（连续变量）和卡方检验（分类变量）。
4.结果显示：糖尿病组腰围、年龄、总胆固醇均显著高于非糖尿病组（p<0.001），性别分布无显著差异。
