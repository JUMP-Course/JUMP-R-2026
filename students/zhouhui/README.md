# 基于UKB数据库研究女性妊娠期糖尿病与冠心病发病风险的关联强度分析
## 作者：周慧
## 课程：JUMP-R-2026 R语言项目实践课

---

## 1. 研究背景
妊娠期糖尿病（GDM）是妊娠期最常见的并发症之一，越来越多的研究表明GDM病史会增加女性远期心血管疾病的发病风险。本研究旨在基于大样本的英国生物银行（UKB）队列数据，量化GDM病史与女性远期冠心病发病风险之间的关联强度。

## 2. 研究问题
有妊娠期糖尿病病史的女性，远期发生冠心病的风险是否显著高于无妊娠期糖尿病病史的女性？

## 3. 数据来源
英国生物银行（UK Biobank, UKB）公开队列研究的精简子集，文件名为`test.rds`，共纳入273315名研究对象。

## 4. 核心变量
- **暴露变量**：妊娠期糖尿病史（gdm），二分类，1=有GDM病史，0=无GDM病史
- **结局变量**：冠心病发病（chd），二分类，1=发生冠心病，0=未发生冠心病
- **协变量**：年龄（age）、性别（sex）、体重指数（bmi）、吸烟状态（smoking）、教育程度（education）、饮酒状况（alcohol）
